Arquitectura

Diseñada para durar

Tecnología moderna que garantiza rendimiento, seguridad y escalabilidad.

Stack tecnológico

Next.js 16 + Supabase + Tailwind CSS 4 — un ecosistema probado y en constante evolución.

# ── Frontend ──
Framework: Next.js 16 (App Router · React 19)
UI: Tailwind CSS 4 · shadcn/ui · Lucide Icons · Recharts · Leaflet
Estado: Context API (Theme · Role · Alert · Toast)

# ── Backend ──
Base de datos: PostgreSQL 15 (Supabase)
Auth: Supabase Auth (Row Level Security · Roles admin/user)
API: Server Actions · Route Handlers · Edge Functions
Storage: Supabase Storage (fotos · documentos)

# ── Calidad ──
Tests: Vitest · React Testing Library
Validación: Zod schemas por módulo
Despliegue: Vercel (CI/CD automático · Edge Network)
Notificaciones: Resend (email transaccional)
Seguridad

Protegemos tus datos

Cada capa de la plataforma está diseñada con seguridad desde su arquitectura.

✓

Autenticación segura

Supabase Auth con sesiones JWT, refresh tokens y protección contra CSRF.

✓

Row Level Security

Políticas a nivel de fila en PostgreSQL: cada usuario accede solo a sus datos autorizados.

✓

Server Actions

Toda operación de escritura se ejecuta del lado del servidor, nunca expone lógica al cliente.

✓

HTTPS + Edge Network

Despliegue en Vercel con CDN global, cifrado TLS y mitigación DDoS incluida.

✓

Validación con Zod

Schemas de validación estrictos en cada operación de entrada y salida de datos.

✓

Storage seguro

Archivos y documentos almacenados en Supabase Storage con políticas de acceso granulares.
